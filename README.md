# ひっぱるこうかとんストライク

## 実行環境の必要条件
* python >= 3.10
* pygame >= 2.1
* 必要なものがあれば追記してください（非推奨）

## ゲームの概要
* 主人公キャラクター「こうかとん」をマウス操作で引っ張って弾き、敵を倒してステージを進めていくアクションゲームです。
* 参考URL：[サイトタイトル](https://www.hoge.com/)

## ゲームの遊び方
* こうかとんをマウスカーソルで引っ張ってはじく


## ゲームの実装
* 背景とキャラクターの描画： 戦場（senjou.png）の背景描画、敵の描画、およびこうかとんの描画。

* 物理挙動・操作： マウスドラッグによる引っ張り制限（最大200px）と、ガイドライン（予測線）の描画。壁での跳ね返りおよび摩擦による減速処理。

* サウンド： ゲームプレイ中のBGM（bgm.mp3）の無限ループ再生。
 
### 共通基本機能
* 背景画像と敵、主人公キャラクターの描画

### 分担追加機能
* 衝突時の多段ヒット防止　エフェクト　C0A25220
* ギミックの追加（障害物、敵の追加）  C0A25224  
* 愛情コンビネーション              C0A25248
* ゲームクリア・オーバー画面の追加   C0C25022   

###　担当追加機能
* 自身のHPがなくなればゲームオーバー
* 敵のHPがなくなればゲームクリア

### メモ
* クラス内の変数は，すべて，「get_変数名」という名前のメソッドを介してアクセスするように設計してある
* すべてのクラスに関係する関数は，クラスの外で定義してある
